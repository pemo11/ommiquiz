# About OMMI Quiz

OMMI Quiz helps students review key concepts with curated flashcards. It offers a quick way to pick a deck, study the cards, and switch to admin mode for managing content.

## Features
- Browse available flashcard sets and load them instantly.
- Study each card with a clean, distraction-free viewer.
- Toggle to admin mode to upload or update quiz content.
- See the current frontend version directly in the header.
